Project overview

Book Explorer

A Phase 1 capstone project that combines responsive layout, API data, and browser storage in a simple book discovery experience.

01

Responsive interface

The layout adapts cleanly from mobile to desktop, with readable spacing and predictable navigation.

02

API-powered results

Search results come from the Open Library API using asynchronous JavaScript and dynamic rendering.

03

Persistent favorites

Favorite books are stored with localStorage, so selections remain after refreshing the page.

04

Organized modules

The JavaScript is separated by responsibility, making the app easier to understand and maintain.
